ggfortify
=========

Define ``fortify`` and ``autoplot`` functions to allow ``ggplot2`` to handle some popular R packages.

This covers following classes:

- ``base::matrix``
- ``base::table``
- ``changepoint::cpt``
- ``forecast::forecast``
- ``stats::acf``
- ``stats::ar``
- ``stats::Arima``
- ``stats::decomposed.ts``
- ``stats::factanal``
- ``stats::HoltWinters``
- ``stats::kmeans``
- ``stats::prcomp``
- ``stats::princomp``
- ``stats::spec``
- ``stats::stl``
- ``stats::ts``
- ``survival::survfit``
- ``survival::survfit.cox``
- ``timeSeries::timeSeries``
- ``tseries::irts``
- ``vars::varprd``
- ``xts::xts``
