ggfortify
=========

Define `fortify` functions to allow `ggplot2` to handle some popular R packages.

This covers following classes:

- ``changepoint::cpt``
- ``forecast::forecast``
- ``stats::stl``
- ``stats::acf``
- ``stats::spec``
- ``survival::survfit``
- ``survival::survfit.cox``
- ``vars::varprd``
